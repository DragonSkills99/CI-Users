Whitelist and Blacklist

The whitelist allows you to define domains of E-Mails which should be allowed, if anyone with another domain tries to register, he will get an error. (The domain is anything after the "@")

The blacklist allows you to define domains of E-Mails which should be disallowed, if anyone with a defined domain tries to register, he will get an error.

Whitelist

To create a whitelist you just have to fill the $email_whitelist array in the Users Controller.

e.g. "protected $email_whitelist = array( "gmail.com" );"

any E-Mail-Address instead of @gmail.com will be disallowed.

Blacklist

To create a blacklist you just have to fill the $email_blacklist array in the Users Controller.

e.g. "protected $email_blacklist = array( "gmx.net" );"

any @gmx.de E-Mail-Address will be disallowed.
